# MISUFoundation

MISUFoundation是一个针对OS X和iOS软件开发的轻量级基础框架, 叫做MISUFoundation的原因是因为爱. 其基本的目标是:

* 尽可能 简洁 安全 的完成基础任务

## 模块
___

* Memory 内存
* Network 网络
